# 太平年连连看

基于电视剧《太平年》角色创作的微信小游戏。

## 🎬 关于《太平年》

电视剧《太平年》讲述五代十国末期，吴越国主钱弘俶"纳土归宋"实现和平统一的历史故事。

## 🎮 游戏特色

- **AI生成角色** - 使用Gemini AI生成的12个角色卡通形象
- **角色图鉴** - 查看所有角色信息和演员介绍
- **三种难度** - 简单(4×4)、普通(6×6)、困难(8×8)
- **经典玩法** - 最多3条直线连接的消除规则
- **道具系统** - 提示和重排功能

## 📱 操作说明

1. 点击两个相同的角色头像进行消除
2. 连接路径不能超过3条直线（0-2个拐角）
3. 点击已选中的角色可查看详细信息
4. 在时间内消除所有卡片获胜

## 🚀 快速开始

1. 下载[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目 `~/peace-llk/`
3. 使用测试号即可运行
4. 点击「真机调试」扫码手机测试
