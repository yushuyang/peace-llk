# 太平年连连看 - 微信开发者工具测试指南

## 📥 第一步：下载安装微信开发者工具

1. 访问官方下载页面：
   ```
   https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   ```

2. 下载对应系统的稳定版（Stable Build）
   - macOS: `wechat_devtools_x.x.x.dmg`
   - Windows: `wechat_devtools_x.x.x.exe`

3. 安装并打开开发者工具

## 📂 第二步：导入项目

### 方法1：通过项目配置导入（推荐）

1. 点击「导入项目」
2. 选择项目目录：`~/peace-llk/`
3. AppID 选择「使用测试号」（自动生成测试账号）
4. 点击「确定」

### 方法2：手动创建

1. 点击「+」创建新项目
2. 项目名称：太平年连连看
3. 目录：`~/peace-llk/`
4. AppID：点击「注册」获取测试号，或使用 `wxtestappid`
5. 开发模式：小程序
6. 点击「创建」

1. **图片加载测试**
   - 观察角色图片是否正常显示

2. **内存测试**

## 📱 第四步：真机调试

2. 选择「编译并自动调试」
3. 使用微信扫描二维码
4. 手机端自动打开小程序
5. 电脑端显示Console日志

### 真机测试要点

- 图片加载速度
- 动画流畅度
- 不同机型适配

## 🔧 常见问题排查

### 问题1：图片无法显示

**现象**：游戏格子显示空白或裂图图标

**排查**：
1. 检查Console是否有404错误
2. 确认图片路径正确：`/images/char01.png`
3. 检查图片文件是否存在：`ls ~/peace-llk/images/`

### 问题2：ES6模块导入错误

**现象**：`Cannot use import statement outside a module`

**解决**：
1. 打开「详情」→「本地设置」
2. 勾选「增强编译」
3. 设置「ES6转ES5」

### 问题3：Canvas不显示

**现象**：游戏板空白

**解决**：
1. 检查基础库版本（设置→项目设置→调试基础库）
2. 确保选择2.14.4以上版本

### 问题4：数据不保存

**现象**：最高分不记录

**解决**：
1. 使用「预览」或「真机调试」模式
2. 模拟器不支持localStorage持久化
