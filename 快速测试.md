## ⚡ 5分钟快速测试流程

### 1. 启动测试 (30秒)
- [ ] 打开微信开发者工具
- [ ] 导入 `~/peace-llk/` 项目
- [ ] 等待编译完成，无报错

### 2. 首页测试 (30秒)
- [ ] 页面显示正常
- [ ] 点击「开始游戏」→ 进入游戏
- [ ] 点击「角色图鉴」→ 进入图鉴

### 3. 角色图鉴测试 (1分钟)
- [ ] 显示12个角色
- [ ] 所有图片加载正常（无裂图）
- [ ] 点击角色 → 显示详情弹窗
- [ ] 返回首页

### 4. 游戏测试 (2分钟)
- [ ] 选择「简单」难度 → 开始游戏
- [ ] 检查4×4格子显示正常
- [ ] 点击角色 → 显示金色边框
- [ ] 点击已选角色 → 显示详情弹窗
- [ ] 找到两个相同角色 → 点击消除
- [ ] 检查得分增加
- [ ] 点击「提示」→ 查看提示效果
- [ ] 点击「暂停」→ 暂停/继续

### 5. 完成游戏 (1分钟)
- [ ] 消除所有角色 → 查看胜利界面
- [ ] 或等待时间结束 → 查看失败界面
- [ ] 点击「再来一局」→ 重新开始

## 🐛 常见问题速查

### 图片不显示
```
检查 Console → 是否有 404 错误
检查 images/ 目录 → 是否有 12 张图片
```

### 游戏板空白
```
检查基础库版本 → 设置 → 项目设置 → 调试基础库 → 2.14.4+
检查 Console → 是否有 JS 错误
```

### 无法消除
```
检查两个角色是否真的可以连接（最多3条直线）
检查 Console → path 是否为 null
```

### 最高分不记录
```
正常现象！模拟器不支持 localStorage
使用「预览」或「真机调试」模式
```
