# 微信开发者工具测试指南

## 🚀 开始测试

### 第一步：启动微信开发者工具

1. 打开 **微信开发者工具**
2. 点击 **「导入项目」**
3. 选择目录：`~/peace-llk/`
4. AppID 选择 **「使用测试号」**
5. 点击 **「确定」**

### 第二步：等待项目加载

- 首次加载可能需要 10-30 秒
- 观察 Console 面板是否有报错
- 确认项目成功编译

## ⚠️ 常见问题排查

### 问题 1：图片无法显示

**现象**：游戏格子显示空白或裂图图标

**排查步骤**：
1. 打开 Console 面板
2. 检查是否有 404 错误
3. 确认图片路径正确：`/images/char01.png`
4. 检查图片文件是否存在

**解决**：
```bash
ls ~/peace-llk/images/char*.png
```

### 问题 2：游戏板不显示

**现象**：游戏页面空白，没有格子

**排查**：
1. 检查 Canvas 是否正确初始化
2. 查看 Console 是否有 JavaScript 错误
3. 确认基础库版本 2.14.4 以上

### 问题 3：无法消除

**现象**：点击两个相同角色没有反应

**排查**：
1. 检查 `LinkUpGame.js` 是否正确加载
2. 检查 `findPath` 算法是否正常工作
3. 在 Console 中打印 path 结果调试

### 问题 4：最高分不保存

**现象**：游戏结束后最高分没有更新

**原因**：模拟器不支持 localStorage 持久化

**解决**：使用「预览」或「真机调试」模式测试
